## Setup

### 1) Clone
```bash
git clone https://github.com/<your-org-or-user>/DocMagic.git
cd DocMagic
```
### 2) Install

```bash
pnpm install   # or: npm install
```
### 3) Configure environment (see .env.example below)
```bash
cp .env.example .env.local
```
### 4) Run
```bash
npm dev  or npm run dev
```

### 5) Build / Start
```bash
npm build && npm start
```
